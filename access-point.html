ACCESS POINT

Urządzenie rozszerzające sieć bezprzewodową WiFi.

Czym jest access point?

Access Point to urządzenie sieciowe, które umożliwia tworzenie bezprzewodowej sieci WiFi.

Najczęściej używany jest do rozszerzania zasięgu sieci w domach, biurach i szkołach.

Jak działa access point?

Odbieranie internetu

Access Point pobiera połączenie z routera lub switcha.

Tworzenie WiFi

Urządzenie tworzy sieć bezprzewodową dla użytkowników.

Łączenie urządzeń

Smartfony i komputery mogą połączyć się z siecią WiFi.

Zalety i wady

Zalety

  • Większy zasięg WiFi
  • Obsługa wielu urządzeń
  • Stabilne połączenie
  • Łatwa rozbudowa sieci

Wady

  • Wymaga routera
  • Może tracić sygnał
  • Lepsze modele są droższe
